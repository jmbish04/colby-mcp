Cloudflare MCP Worker

This Worker exposes a Model Context Protocol server with Cloudflare-native tools including Browser Rendering, D1, KV, Vectorize, and Durable Objects.

Use the /mcp (WebSocket) or /sse (Server Sent Events) endpoints to connect an MCP-compatible client, or call the REST endpoints under /api/tools to invoke tools directly.

An OpenAPI specification describes the REST surface for easy integration with assistants like Claude, Copilot, Gemini, or Cursor.

  • WebSocket: /mcp
  • Server-Sent Events: /sse
  • REST invocation: /api/tools/<toolName>
